# 💰 Finance Micro Tools

A modern, lightweight financial dashboard built with **vanilla JavaScript, HTML, and CSS** that provides powerful tools for budgeting, debt payoff analysis, investment growth, and subscription tracking.

---

## 🚀 Features

### 📊 Budget Planner

* Compare income vs expenses
* Visual breakdown using Chart.js
* Automatically saves data with localStorage

### 💳 Debt Payoff Calculator

* Calculates payoff timeline (in months)
* Generates a full **amortization schedule**
* Displays:

  * Monthly interest
  * Principal paid
  * Remaining balance
* Export data to **CSV**

### 📈 Compound Interest Calculator

* Visualizes investment growth over time
* Interactive line chart powered by Chart.js

### 🔁 Subscription Tracker

* Dynamically add/remove subscriptions
* Calculates:

  * Total monthly cost
  * Yearly spending

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/C-M1nusM1nus/finance-micro-tools.git
cd finance-micro-tools
```

### 2. Open locally

Just open `index.html` in your browser.

---

## 🌐 Deploy with GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings**
3. Navigate to **Pages**
4. Under **Source**, select:

   * Branch: `main`
   * Folder: `/root`
5. Click **Save**

Your site will be live at:

```
https://your-username.github.io/finance-micro-tools/
```
